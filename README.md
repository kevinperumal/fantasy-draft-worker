# DraftPilot — Worker

This repository contains the **automation worker** for DraftPilot.

The worker logs into a fantasy draft room, injects automation logic, and monitors the draft in real time.

➡️ **Project overview:** https://github.com/kevinperumal/draftpilot

## Responsibilities
- Log into the draft room
- Inject and run draft automation logic
- Monitor draft state and events
- Communicate status and data back to the backend API

## Tech stack
TypeScript, Node.js

## Notes
Secrets and credentials are managed via environment variables and are not committed to the repository.
